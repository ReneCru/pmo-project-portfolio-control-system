# PMO Project Portfolio Control System

A professional PMO analytics project designed to monitor and control a portfolio of projects using Python, SQL, and Power BI.

## Project Objective

The objective of this project is to build a Project Portfolio Control System that allows a PMO, Project Manager, or Operations team to track project performance, budget variance, milestone delays, risk exposure, and executive-level KPIs across multiple business areas.

## Business Context

Organizations often manage multiple projects at the same time across departments such as Supply Chain, Operations, IT, Quality, Finance, and Engineering. Without a centralized control system, leadership may struggle to identify delayed projects, budget overruns, unmanaged risks, and priority conflicts.

This project simulates a PMO control environment where project data is consolidated, analyzed, and visualized to support better decision-making.

## Key Features

* Project portfolio tracking
* Budget variance analysis
* Milestone delay monitoring
* Risk and issue management
* Executive KPI dashboard
* Department and project manager performance views
* Data cleaning and analysis using Python
* SQL queries for portfolio insights
* Power BI dashboard for executive reporting
