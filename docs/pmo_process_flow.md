# PMO Process Flow: Project Portfolio Control System

## 1. Purpose

This document describes the PMO process flow used in the Project Portfolio Control System.

The objective is to define how projects move through the portfolio control process, from initial registration to executive reporting and corrective action tracking.

This process supports better visibility, accountability, project governance, and decision-making across the organization.

---

## 2. Process Overview

The PMO Project Portfolio Control System follows this process flow:

```text
Project Intake
→ Project Approval
→ Portfolio Registration
→ Project Planning
→ Milestone Tracking
→ Budget Monitoring
→ Risk and Issue Review
→ KPI Calculation
→ Executive Dashboard Reporting
→ Corrective Actions
→ Project Closure
