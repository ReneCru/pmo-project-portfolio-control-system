# Power BI Build Steps: PMO Project Portfolio Control System

## 1. Purpose

This document provides step-by-step instructions to build the Power BI dashboard for the PMO Project Portfolio Control System.

The dashboard is designed to visualize project portfolio health, budget performance, risk exposure, milestone delays, and executive attention priorities.

---

## 2. Prerequisites

Before building the Power BI dashboard, run the following scripts successfully:

```bash
python src/generate_sample_data.py
python src/clean_portfolio_data.py
python src/portfolio_kpi_analysis.py
